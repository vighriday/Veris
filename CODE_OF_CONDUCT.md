# Code of Conduct

We follow the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## In short

Be excellent to each other. Be technically rigorous. Disagree on ideas, not people. Respect maintainer time.

## Reporting

Conduct concerns: open a private GitHub Security Advisory or DM the maintainer on the contact channel listed in the README. Maintainer responses target 72 hours.

## Enforcement

The maintainer will use the enforcement guidelines in the Contributor Covenant v2.1.
